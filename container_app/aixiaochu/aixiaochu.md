##游戏说明

游戏界面模仿微信游戏天天系列中的——天天爱消除

界面素材和声音素材来源于网上

##源代码说明
```
下了代码不能运行的朋友请先看这里：
 + 1、游戏需要在支持html5的浏览器中运行
 + 2、游戏需要在web服务器环境下运行(IIS、apache、nginx等)
 + 3、本地测试请将game.js中的debug参数置为true
```
